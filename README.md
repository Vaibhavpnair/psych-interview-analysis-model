# Psychiatric Interview Assistance System

A multimodal AI decision-support system for psychiatrists.

## Modules
- **Vision**: Facial expression & micro-expression analysis.
- **Audio**: Speech prosody, pause detection, & pitch analysis.
- **NLP**: Linguistic marker detection (Absolutist words, Sentiment).
- **Fusion**: Hybrid risk scoring based on multimodal inputs.

## Stack
- **Backend**: FastAPI (Python)
- **Frontend**: React (Vite + Tailwind)
- **AI**: OpenAI Whisper, MediaPipe, spaCy, Transformers
